В данной программе моделериуется процесс формирования OFDM сигнала, осуществляется его передача через двулучевой канал связи, производится эквализация и оценка канала с помощью пилотного символа.
В дальнейшем планируется перейти от SISO к MIMO модели и построить более сложный канал связи, используя спецификацию TR 38.901
